## Step 6 — Add a launcher icon
  Android Studio → right-click res → New → Image Asset
  Choose any camera icon. This generates mipmap/ic_launcher*.

## Step 7 — Build & Run
  Connect Android device via USB → Run 'app'
  Grant Camera + Notification permissions when prompted.
  Tap "Open Settings" on the battery optimization dialog.

## Notes
- The two _README files in res/drawable/ are documentation only, not real drawables.
  Delete them before building (or Android Studio will warn about unknown resource format).
- activity_main.xml references androidx.camera.view.PreviewView which comes from
  the CameraX view dependency already in build.gradle.kts.
- themes.xml uses Theme.MaterialComponents — requires the Material dependency in gradle.
